# Kontroli
Sistema de controle de estoque e vendas desenvolvido em React, com integração a banco de dados Neon (PostgreSQL). Permite cadastro de produtos, controle de estoque mínimo, registro de vendas e geração de relatórios em tempo real, oferecendo uma solução simples, eficiente e escalável para pequenos e médios negócios.


📌 **Sobre o projeto

O Kontroli é um sistema web Fullstack que permite o gerenciamento eficiente de estoque e vendas em tempo real.
Desenvolvido como Trabalho de Conclusão de Curso (TCC), o projeto simula um ambiente real de negócio, aplicando boas práticas de desenvolvimento e arquitetura moderna.

---

✨ Funcionalidades

📦 Gestão de Estoque

* Cadastro de produtos
* Organização por categorias
* Definição de estoque mínimo
* Alerta automático de produtos com baixo estoque

💰 Vendas

* Registro de vendas com poucos cliques
* Seleção de produto, quantidade e vendedor
* Atualização automática do estoque

📊 Relatórios Inteligentes

* Faturamento do dia
* Vendas do dia
* Faturamento do mês
* Vendas do mês
* Tabela completa com histórico de vendas

🔎 Filtros e Controle

* Filtro por categorias
* Visualização rápida e organizada

---

🧠 Diferenciais

* Interface limpa e intuitiva
* Atualização em tempo real
* Estrutura escalável
* Código componentizado (React)
* Pensado para resolver problemas reais de pequenos negócios

---

 🛠️ Tecnologias Utilizadas

* ⚛️ React — Front-end moderno e dinâmico
* 🌐 HTML5 & CSS3 — Estrutura e estilização
* 📜 JavaScript — Lógica da aplicação
* 🗄️ Neon (PostgreSQL) — Banco de dados em nuvem
* 🔗 API — Comunicação entre front e banco
* 🧪 Postman — Testes de requisições

---

🔄 Arquitetura do Sistema

```
React (Front-end)
        ↓
API / Back-end
        ↓
Neon (Banco de Dados em Nuvem)
        ↑
Postman (Testes de API)
```

💻 Como rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/kontroli

# Acesse a pasta
cd kontroli

# Instale as dependências
npm install

# Rode o projeto
npm start
```

---

⚠️ Como executar o projeto

Para o funcionamento correto do sistema, é necessário rodar o front-end e o back-end separadamente.

O projeto está dividido em duas pastas:

📁 Origin-Kontroli-React → Front-end (React)
📁 server** → Back-end (API)

---

▶️ Passo a passo

Abra duas janelas do VS Code:

---

💻 1. Rodar o Front-end

```bash
cd Origin-Kontroli-React
npm install
npm start
```

---

⚙️ 2. Rodar o Back-end

```bash
cd server
npm install
npm run dev
```

---

💡 Importante:
As duas aplicações precisam estar rodando ao mesmo tempo para que o sistema funcione corretamente, garantindo a comunicação entre interface, API e banco de dados.
